# 🚀 Projects 

### 🛍️ Visual Product Similarity Search & Personalized Recommendation System

**Project Overview:** This project implements an end-to-end e-commerce recommendation engine that provides highly accurate visual similarity search and personalized product suggestions. It addresses the cold-start problem and leverages multimodal data (images and text) for a robust customer experience.

**Tech Stack:** `Python`, `PyTorch/TensorFlow`, `CLIP (Contrastive Language-Image Pre-training)`, `Sentence-BERT`, `FAISS (Vector Search)`, `Streamlit/Dash (for UI)`, `Pandas/NumPy`.

**Key Features & Implementation:**

* **Multimodal Embedding Generation:** Utilized **CLIP** for generating joint image-text vector representations and **Sentence-BERT** for enhanced text metadata embeddings.
* **Vector Search Engine:** Implemented **FAISS** for highly efficient, low-latency similarity search across a large corpus of product vectors.
* **Comprehensive EDA:** Conducted full EDA (category distribution, metadata quality, image analysis) to inform feature engineering and model design.
* **Interactive Visualization:** Developed an interactive dashboard to visualize product clusters, similarity relationships, and model performance metrics.
* **Deployment Ready:** Architected as a modular ML project ready for API exposure and integration into a production environment.

**How to Run/Replicate:**
1.  **Dependencies:** Clone the repository and install required packages (`pip install -r requirements.txt`).
2.  **Dataset Note:** Due to its size, the raw e-commerce dataset must be downloaded separately (link provided in the project repository).
3.  **Setup:** Run the embedding generation notebook (`/notebooks/generate_embeddings.ipynb`) to create the vector index.
4.  **Usage:** Execute the main application file (`python run_app.py`) to launch the interactive UI for search and recommendations.
