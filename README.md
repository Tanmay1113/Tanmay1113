### **🛍️ Visual Product Similarity Search & Personalized Recommendation Prototype**

**Focus:** Comprehensive application of Deep Learning, NLP, and efficient data retrieval techniques for an e-commerce platform prototype.

**Core Tech Stack Highlighted:** `Python`, **`PyTorch`**, **`CLIP (Vision-Language)`**, **`Sentence-BERT`**, **`FAISS`**, `Docker`, `Streamlit (UI)`.

**Key Technical Contributions (Focusing on Fundamentals):**

* **Deep Multimodal Feature Extraction:** Successfully implemented and utilized the **CLIP** model for generating joint, high-quality vector representations from both product images (Computer Vision) and textual descriptions (NLP).
* **High-Speed Data Retrieval:** Built a lightning-fast nearest neighbor index using **FAISS**, drastically reducing lookup time for product similarity search.
* **Project Containerization:** Created a prototype environment using **Docker** to ensure all dependencies were packaged consistently and the application ran smoothly.
* **Data Processing Pipeline:** Performed extensive **EDA** and data wrangling with Pandas/NumPy, structuring the raw data for model training and feature engineering.
* **Demonstration:** Developed an interactive **Streamlit** front-end to showcase the recommendation results and similarity search functionality.
