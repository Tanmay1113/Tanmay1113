## üöÄ Featured Project: Real-Time Credit Card Fraud Detection

### **üîê End-to-End Machine Learning System**

**Focus:** Detecting fraudulent financial transactions in highly imbalanced datasets with real-time inference capability.

**Tech Stack:**  
`Python`, `Pandas`, `Scikit-learn`, `Random Forest`, `SMOTE`, `FastAPI`, `Docker`

**Key Contributions:**
* Built a complete **ML pipeline** including EDA, feature engineering, and model training.
* Addressed severe class imbalance using **SMOTE**, improving fraud **recall to ~90%**.
* Tuned decision thresholds to prioritize **recall over accuracy**, aligned with real-world fraud prevention.
* Simulated **real-time transaction processing** with instant Block / Allow decisions.
* Deployed the trained model as a **FastAPI REST service** for low-latency predictions.
